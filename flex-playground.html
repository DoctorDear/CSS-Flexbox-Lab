CSS Flex Container Properties

display flex-direction flex-wrap flex-flow justify-content align-items Playground

display 🎬

จุดเริ่มต้นของทุกอย่าง — เปลี่ยนให้ element กลายเป็น flex container ก่อน ถึงจะใช้ property อื่นได้

justify-content ↔️

    
จัด items ตาม main-axis (แกนหลัก = แนวเรียงของ items) เมื่อมีพื้นที่เหลือ

      เปรียบเทียบ:

      • space-between — ช่องว่างเฉพาะ ระหว่าง items (ขอบสุดชิดขอบ)

      • space-around — แต่ละ item มีช่องว่าง รอบตัว (ขอบนอกครึ่งหนึ่ง)

      • space-evenly — ช่องว่าง เท่ากันหมด รวมขอบนอก

จัด items ตาม cross-axis (แกนตั้งฉาก = แนวที่ตั้งฉากกับการเรียง) เช่น กรณี row คือแนวตั้ง
