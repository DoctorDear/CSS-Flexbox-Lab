# 🧪 CSS Flexbox Lab

**CSS Flexbox Lab** คือโปรเจกต์สำหรับเรียนรู้และทำความเข้าใจการจัดหน้าเว็บด้วย CSS Flexbox อย่างลึกซึ้งผ่านการลงมือปฏิบัติจริง (Interactive Learning) เหมาะสำหรับผู้เริ่มต้นที่ต้องการเห็นภาพการทำงานของ Flexbox ในแต่ละแกน หรือนักพัฒนาที่ต้องการทบทวนความรู้

## ✨ ฟีเจอร์เด่น (Features)

- **Interactive Playground**: ทดลองเปลี่ยนค่า Properties ต่างๆ ของ Flexbox (เช่น `justify-content`, `align-items`, `flex-direction`) และเห็นผลลัพธ์การจัดเรียงกล่องแบบ Real-time
- **Cheat Sheet & Visual Cards**: การ์ดสรุปคุณสมบัติแต่ละค่า พร้อมภาพประกอบขนาดเล็กที่เข้าใจง่าย (ช่วยให้จำได้ใน 30 วินาที)
- **แยกหมวดหมู่ชัดเจน**: อธิบายเจาะลึกแบบทีละเรื่อง ตั้งแต่พื้นฐาน `display: flex` ไปจนถึงการจัดวางแบบ Advance
- **Light / Dark Mode**: รองรับการสลับธีมสว่างและมืดเพื่อความสบายตาระหว่างเขียนโค้ด
- **No Dependencies**: พัฒนาด้วย Vanilla HTML, CSS (Modern CSS Grid & Custom Properties) และ JavaScript ล้วนๆ ไม่ต้องใช้ Library หรือ Framework ใดๆ ทำงานได้รวดเร็วทันที

## 📂 โครงสร้างไฟล์ (File Structure)

โปรเจกต์นี้ประกอบด้วยไฟล์ HTML ที่แยกตามหัวข้อการเรียนรู้:

- `index.html` - หน้าแรก (สารบัญ และ Master Playground สำหรับลองเล่นรวมทุกคำสั่ง)
- `display.html` - พื้นฐานการเปิดใช้งาน `display: flex` และ `inline-flex`
- `flex-direction.html` - การกำหนดทิศทางของแกนหลัก (Main Axis)
- `flex-wrap.html` - การจัดการเมื่อไอเท็มล้นกล่อง (การปัดบรรทัด)
- `flex-flow.html` - Shorthand ที่รวม `flex-direction` และ `flex-wrap` ไว้ในบรรทัดเดียว
- `justify-content.html` - การจัดเรียงพื้นที่ว่างในแนวแกนหลัก
- `align-items.html` - การจัดเรียงพื้นที่ว่างในแนวแกนรอง (Cross Axis)

## 🚀 วิธีการใช้งาน (How to Use)

เนื่องจากเป็นโปรเจกต์เว็บแบบ Static 100% จึงไม่ต้อง Build หรือ Install อะไรเลยครับ:
1. Clone โปรเจกต์นี้ หรือดาวน์โหลดเป็นไฟล์ ZIP ไปไว้ในเครื่อง
2. ดับเบิ้ลคลิกเปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ใดก็ได้ (Chrome, Firefox, Safari, Edge)
3. เริ่มคลิกเล่นตามปุ่มต่างๆ เพื่อเรียนรู้ Flexbox ได้ทันที!
