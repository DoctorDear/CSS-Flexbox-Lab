05 ·

จุดที่คน พลาดบ่อย ⚠️

① ลืม display: flex แล้ว property อื่น "เงียบหาย"

ตั้ง justify-content, align-items, gap, flex-grow ไป แต่ดันไม่ทำงาน — สาเหตุคือลืมตั้ง display: flex บน container ก่อน property ทุกตัวของ Flexbox ต้องการจุดเริ่มตัวนี้

② inline-flex vs flex — ความกว้างต่างกัน

flex กินความกว้าง เต็มบรรทัด (100% ของ parent) ส่วน inline-flex กิน แค่พอดีเนื้อหา ถ้าเจอปัญหากล่อง "กว้างเกินไป" หรือ "แคบไป" ให้ตรวจดูค่านี้ก่อนเลย

③ flex ใช้กับ container ไม่ใช่ items

คนมักสับสนตั้ง display: flex บนลูกแล้วคิดว่าจะจัดเรียงตัวเอง — ไม่ใช่! display: flex ตั้งบน พ่อแม่ (container) เพื่อคุมลูก ส่วน flex-grow, order ตั้งบน ลูก แยกกัน
